| $AB$ | 自A至B的直线段 |
| $\overline{AB}$ | AB线段的长（有时也用AB表示） |
| $\angle$ | 平面角 |
| $\circ$ | 度 （例 $21^\circ$） |
| $'$ | 分 （例 $21^\circ23'$） |
| $''$ | 秒 （例 $21^\circ23'18''$） |
| $\stackrel\frown{AB}$ <!-- 或用 $\overset\frown{AB}$ --> <!-- 之前用的 $\overparen {AB}$ -->| 弧 |
| $\pi$ | 圆周率 |
| $\triangle$ | 三角形 |
| 无此符号 | 平行四边形 |
| 无此符号 | 矩形 |
| $\perp$ <!-- (or $\bot$ ?) -->| 垂直 | 
| $\parallel$ | 平行 (国内的写法应该是形如：// ，但找不到对应的符号) |
| $\sim$ | 相似 |
| $\cong$ | 全同 |
| $\because$ | 因为 |
| $\therefore$ | 所以 |
